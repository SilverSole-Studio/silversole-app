# DESIGN.md — UI 復刻規格書（Replication Spec）

> 目標：**只看本文字檔，就能 1:1 復刻 `ui_ref/` 內 8 張截圖（Light×4 / Dark×4，共 6 個畫面狀態）**。
> 因此本檔不只描述「風格」，而是描述「每個畫面有什麼、放在哪、多大、什麼顏色、什麼字」。
>
> 參考來源：台灣電子發票 / 雲端發票 + 集點兌禮 App。風格屬現代 fintech / 工具型產品。
>
> ⚠️ **量測值說明**：本檔所有 px/dp/字級皆為「從截圖估算」，非從設計檔匯出，故標 `≈`。
> 來源解析度 **1080 × 2400 px**（比例 9:20，Android）。下文尺寸以 **dp** 表示，假設 **3x**（即 1080px = 360dp 寬、800dp 高）。

## 1. 設計語言（Design Language）

- **分類**：Modern fintech / utility。扁平（flat）為主 + 柔和漸層 + 微陰影。
- **骨架原則**：**灰階打底、單色點睛**。介面結構全走中性灰階；彩色只給「品牌 / 可點 / 重要 / 狀態 / 獎勵」。
- **形狀**：高圓角。卡片 `≈20dp`、Banner 圖 `≈16dp`、晶片/按鈕 = pill（全圓）、FAB = squircle（圓角方）。
- **層次**：靠「底色階差 + 圓角 + 輕陰影」，**不靠粗邊框**。
- **雙主題**：真・深色（deep navy/near-black + 放射光暈），非單純反白。
- **強調雙軸**：🔵 藍 = 操作/品牌/連結；🟡 金 = 獎勵/點數。兩者不混用。

## 3. 排版 Token（Typography Scale）

字型：本專案為 **Oxanium**（`lib/core/theme/typograhpy.dart`）。參考圖數字偏粗、近等寬，調性相容。

| 樣式            | 字級 ≈  | 字重    | 用途                                   |
| --------------- | ------- | ------- | -------------------------------------- |
| `display/hero`  | 32–38dp | 800     | 首頁問候三行（Hello！/ 姓名 / 標語）   |
| `title/page`    | 26dp    | 800     | 頁面大標（我的發票、集點兌禮）         |
| `title/section` | 18dp    | 700     | 卡片標題（消費分析）、月份分組標頭     |
| `number/stat`   | 24dp    | 700     | 統計列數字、中獎金額、列表金額         |
| `body/default`  | 15dp    | 400–500 | 內文、列表商家名（商家名 600）         |
| `label/caption` | 13dp    | 400     | 統計 label、meta（日期/載具號/百分比） |
| `label/nav`     | 11dp    | 500     | 底部導覽文字                           |

排版原則：**用「字級 + 字重」拉層級，而非顏色**；次要資訊一律降為 `text/secondary|hint`。金額/數字偏粗、表格數字（tabular）對齊。

## 7. 元件庫（Component Anatomy）

> 每個元件＝結構 + 量測 + 顏色。復刻時逐項對照。

### C1. 品牌 Logo（左上）

- 兩片重疊的**圓角平行四邊形緞帶**交織成風格化「S / 閃電」，左上角一顆小 sparkle ✦。
- Light：`brand/gradient` 藍→青漸層填色。Dark：**白色**（單色）填色。尺寸 `≈28dp`。

### C2. 統計列（Stats Row, S1）

- 三欄等寬：`本月發票 / 金幣 / 票券`。每欄：上 `label/caption` 灰、下 `number/stat` 近黑。
- 欄間 `≈1dp` 淺灰**豎線**分隔（高 `≈28dp`）。左起對齊頁邊距。

### C3. 卡片（Card）

- `surface/card` 底、`radius/card`、`elevation/card`。內距 `space/cardPad`。
- 標題列：左 `title/section`，右 `→`（前往，`≈22dp` outline）。

### C4. 甜甜圈圖（Donut, S1）

- 外徑 `≈250dp`、環寬（stroke）`≈30dp`、置中。
- 分段：`其他` = `data/grayArc`（占 98%）、`餐飲` = `data/orange`（占 2%，位於頂端略偏右），段間留 `≈2dp` 白/底色細縫。
- 中央：大 `7`（`number/stat`×大）+ 下方 `筆消費明細`（`label/caption` 灰）。
- 兩條 **L 形細引導線（leader line）** 拉到環外標註：左上 `2.0 %` / `餐飲`；右下 `98.0 %` / `其他`（百分比在上、類別在下）。
- 卡內底部：`詳細金額` = 次要按鈕（`surface/chip` 底、`text/primary` 字、pill）。

### C5. 篩選晶片列（Filter Chips）

- 橫向可捲動、pill；最右露出半顆暗示可滑。高 `≈36dp`、左右內距 `≈16dp`、間距 `≈10dp`。
- 第 1 顆選中（§2.3/2.4 inverse 規則）；其餘未選灰字。

### C6. 分頁標籤（Tabs）

- 文字 tab 並排（任務/商城；或商城內 為你推薦…另列為晶片）。
- 選中：文字加粗 + 下方 `brand/primary` **底線指示器**（寬約字寬、`≈3dp`）。未選：`text/secondary`。

### C7. 按鈕（Buttons）

| 類型     | 樣式                                                          |
| -------- | ------------------------------------------------------------- |
| 主要 CTA | `brand/primary` 實心 pill + 白字（立即拿好禮 / 立即填寫）     |
| 獎勵 CTA | `reward/gold` 實心 pill + `#1B2430` 深字（可領取 / 免費兌換） |
| 次要     | `surface/chip` 底 + 深字 pill（詳細金額 / 瞭解更多）          |

- 高 `≈40dp`、左右內距 `≈20dp`。

### C8. FAB

- `brand/primary` squircle、`radius/fab`、白 `＋`、`elevation/fab`。尺寸 `≈60dp`，距右/下邊 `≈20dp`。

### C9. 點數徽章（Point Badge, 集點 App bar）

- `reward/coin` 金色圓（內含白 `i`）+ 右側數字（`首頁=金幣值`，此處顯示 37/38）。

### C10. 對獎摘要卡（Summary Card, S4）

- `radius/card`、頂部有**淡藍/淡紫水平 sheen 漸層**（Dark 為暗版 sheen）。
- 第 1 列：`✓ 一般獎已對` `✓ 雲端獎已對`（灰圈勾）＋ 右 `對獎結果 ›`（`brand/primary` 連結）。
- 下：`中獎金額`（`label/caption`）→ 大 `0元`（`number/stat`×大）+ 同列右側灰字 `中 0 張 / 共 5 張`。

### C11. 月份切換列（Month Switcher, S4/S5）

- 置中 `114 年 7-8 月`（ROC 年；雙月為一期），左右 `‹ ›` 灰 chevron 貼邊。

### C12. 發票明細列（Invoice Row, S4）

- 分組標頭：左 `8月`（`title/section`）、右該月合計 `$4,126`（粗）。
- 每列三區塊：
  - **左**（日期塊）：大 `31`（`≈22dp` 700）+ 下 `週日`（`label/caption` 灰）。
  - **中**：商家名（`body` 600，可換行）；第 2 行 `載具` 灰晶片（小）+ 載具號（`text/hint`，等寬感）。
  - **右**：金額（粗、右對齊）。
- 列間 `outline/divider` 細線；不同月份間留較大空白（Dark 為更深的空條）。

### C13. 獎勵任務項（Reward Item, S2）

- 結構＝**整寬 Banner 圖（`radius/banner`，彩色插畫）** + 其下文字塊：
  - 標題（`body` 700，可兩行）。
  - `🎁 reward` 行（禮物 icon + 內容，灰）。
  - `🕐 date` 行（時鐘 icon + 日期，灰）。
  - 右側對齊的 **主要 CTA**（立即拿好禮 / 立即填寫）。
- 特例「每日簽到」卡：金色漸層 banner（左上小 `發票存摺` logo+字、大 `每日簽到`、右側模糊金幣插圖）；下方 `🪙 +1`、`🕐 無期限`、右 `reward/gold` 按鈕 `可領取`。

### C14. 票券卡（Coupon Card, S3）

- **2 欄格**（gutter `≈12dp`）。每張：
  - 上：票券造型圖（`radius/banner`）— 含品牌 logo、折扣大字（`$100 x3張` / `10% +7% OFF`）、角標（`新客限定`/`日本限定`/`CHECK!` 斜角 ribbon）、底部藍 ribbon `INVOS DATA OFFERS`，側邊有票根穿孔/鋸齒缺口。
  - 中：標題（`body` 600，如「唐吉訶德 優惠券」）。
  - 下：整寬 `reward/gold` 按鈕 `免費兌換`。

### C15. 載具條碼列（Carrier Bar, S1 底）

- 緊貼底部導覽上方的**淺色 peek 列**：置中 `▮▮▮ 顯示載具條碼`（條碼 icon + 文字）。
- 其上方有 carousel 把手 `∧` + 頁點 `● ○`。

### C16. 空狀態（Empty State, S5）

- FAB 仍在右下。

## 11. 與現有程式碼的關係（落差，供決策）

- 現況 `lib/app.dart`：`_accentSeed = Color(0xFF2E7D32)`（**綠**），中性面鎖灰階、Light surface `#FCFCFC`、`surfaceTint` 透明（見 memory：accent-only changes, neutrals stay white）。
- **參考品牌色是藍（`brand/primary #2F6BFF` + cyan），非綠。** 既有架構（M3 `ColorScheme.fromSeed` + 灰階中性 + 無 elevation tint）與此風格天生相容。
- **最小落地路徑**：① `_accentSeed` 換藍；② 新增 `reward/gold`、`data/orange`、`state/alert`、`brand/cyan` 等非種子功能色 token；③ 元件圓角/字重/inverse-chip 對齊本檔。中性骨架不動。
